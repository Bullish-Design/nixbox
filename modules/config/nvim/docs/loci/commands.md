# Commands

| Command | Purpose |
|---|---|
| `:LociInit` | Initialize repository state. |
| `:LociHealth` | Run health checks. |
| `:LociDoctor` | Run repository diagnostics. |
| `:LociOpenRoot` | Open repository root. |
| `:LociRepairFrontmatter` | Repair noncanonical frontmatter. |
| `:LociRepairProjectRefs` | Repair project reference fields. |
| `:LociRefresh` | Run strict refresh pipeline. |
| `:LociJobCancel` | Cancel a running background job. |
| `:LociJobList` | List background jobs. |
| `:LociTrailList` | List managed Trails. |
| `:LociTrailCreate` | Create a managed Trail. |
| `:LociTrailSwitch` | Switch active managed Trail. |
| `:LociTrailSave` | Save active Trail through named API. |
| `:LociTrailLoad` | Load a managed Trail through named API. |
| `:LociTrailRename` | Rename a managed Trail. |
| `:LociTrailDelete` | Delete a managed Trail. |
| `:LociTrailExport` | Export Trail display data. |
| `:LociWorkspaceSwitch` | Switch active workspace. |
| `:LociRepositoryOpen` | Open fallback repository workspace. |
| `:LociWorkspaceCreate` | Create workspace. |
| `:LociWorkspaceClone` | Clone workspace. |
| `:LociKnowledgeAdd` | Add knowledge object to workspace. |
| `:LociKnowledgeRemove` | Remove knowledge object from workspace. |
| `:LociKnowledgeSetPrimary` | Set primary knowledge object. |
| `:LociFileLink` | Link repository-relative file to workspace. |
| `:LociFileUnlink` | Unlink repository-relative file from workspace. |
| `:LociWorkspaceInfo` | Show workspace summary. |
| `:LociWorkspaceArchive` | Archive workspace. |
| `:LociWorkspaceRefresh` | Refresh workspace projections. |
| `:LociNoteCreate` | Create note. |
| `:LociDailyNote` | Open/create daily note. |
| `:LociScratchNote` | Create scratch note. |
| `:LociNoteAdopt` | Ensure loci_id on current note. |
| `:LociHauntList` | List Haunt contexts. |
| `:LociHauntNew` | Create Haunt context. |
| `:LociHauntSwitch` | Switch Haunt context. |
| `:LociHauntRename` | Rename Haunt context. |
| `:LociHauntDelete` | Delete Haunt context data. |
| `:LociProjectCreate` | Create project. |
| `:LociProjectOpen` | Open project. |
| `:LociProjectSwitch` | Switch project context. |
| `:LociProjectLink` | Link current object to project. |
| `:LociProjectInfo` | Show project info. |
| `:LociProjectRefresh` | Refresh project projections. |

Loci does not provide compatibility aliases for removed command names.
